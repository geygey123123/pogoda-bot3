# Telegram Weather Bot 🌤️

Асинхронный Telegram-бот для показа подробной погоды с использованием OpenWeatherMap API.

## Возможности

### Основной функционал:
- **Определение локации двумя способами:**
  - Кнопка "Поделиться геопозицией" (отправка координат)
  - Ручной ввод названия города (русский, украинский, английский)

- **Подробная информация о погоде:**
  - Текущая температура и "ощущается как"
  - Описание погодных условий с эмодзи
  - Влажность, давление (мм рт. ст.)
  - Скорость и направление ветра
  - Время рассвета и заката

- **Прогноз на 3 дня** с температурными диапазонами

- **Сохранение настроек** в SQLite базе данных

- **Опциональная синхронизация** с Google Sheets для аналитики

## Структура проекта

```
weather_bot/
```

## Установка

### 1. Клонируйте репозиторий или создайте файлы

### 2. Установите зависимости:

```bash
pip install -r requirements.txt
```

### 3. Настройте переменные окружения:

Скопируйте `.env.example` в `.env` и заполните необходимыми токенами:

```bash
cp .env.example .env
```

Отредактируйте файл `.env`:

```env
# Telegram Bot Token (получить у @BotFather)
TELEGRAM_BOT_TOKEN=your_telegram_bot_token_here

# OpenWeatherMap API Key (бесплатно на https://openweathermap.org/api)
OPENWEATHER_API_KEY=your_openweather_api_key_here

# Google Sheets credentials (опционально)
GOOGLE_CREDENTIALS_JSON=
GOOGLE_SHEET_NAME=weather_bot_users

# Путь к базе данных
DATABASE_PATH=users.db
```

## Получение токенов

### Telegram Bot Token:
1. Откройте @BotFather в Telegram
2. Отправьте команду `/newbot`
3. Следуйте инструкциям для создания бота
4. Скопируйте полученный токен

### OpenWeatherMap API Key:
1. Зарегистрируйтесь на https://openweathermap.org/
2. Перейдите в раздел API Keys
3. Создайте новый ключ (бесплатно)
4. Скопируйте ключ

### Google Sheets Credentials (опционально):
1. Создайте проект в Google Cloud Console
2. Включите Google Sheets API и Google Drive API
3. Создайте Service Account
4. Скачайте JSON файл с учетными данными
5. Вставьте содержимое JSON в `GOOGLE_CREDENTIALS_JSON`

## Запуск бота

```bash
cd weather_bot
python bot.py
```

## Использование

1. Запустите бота командой `/start`
2. Выберите способ определения локации:
   - Нажмите "📍 Поделиться геопозицией" и отправьте координаты
   - Или введите название города текстом (например: "Киев", "Москва", "London")
3. Используйте кнопки меню:
   - **🌤 Погода сейчас** - подробная текущая погода
   - **📅 Прогноз на 3 дня** - прогноз на ближайшие дни
   - **⚙ Настройки** - информация о текущих настройках
   - **📍 Поделиться геопозицией** - изменить локацию

## Команды

- `/start` - Запустить бота
- `/help` - Показать справку

## Особенности

- **Асинхронная архитектура** на базе aiogram 3.x
- **Локальное хранилище** SQLite для сохранения данных пользователей
- **Экспорт метаданных** в Google Sheets для аналитики
- **Мультиязычность** - поддержка русского, украинского и английского языков
- **Красивый вывод** с использованием эмодзи и форматирования

## Требования

- Python 3.8+
- aiogram 3.x
- aiohttp
- python-dotenv
- gspread (опционально, для Google Sheets)
- google-auth (опционально, для Google Sheets)
